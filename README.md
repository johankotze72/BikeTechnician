# BikeTechnician
A tool to enable anyone around the world to be(come) a successful bike mechanic.
This is an initiative started by [DAZUD](www.dazud.com), a non-profit using bikes to make a difference in peoples lifes. After importing and redistributing numerous used bikes to the underprevilidged in South Africa, we realized the growing demand for qualified bike mechanics. It is a very useful skill set, enabling an entepreneur to start his/her own business with very little caplital, or to apply as a technician in larger bike shops. 

## HOW
Develop an open-sourced app using Flutter for multi-operability. The app should be the technicians daily companion for learning/improving his/her bike repair knowledge/skills and help turn it into a profitable bussiness. 

### Learning (Phase I)
Teach fix a bike as easily and entertaining as possible, using modern mobiles methods. Can be uses as stand-alone or as companion to official coarses.
- **Flashcards** (Spaced repetition cards for parts, tools, types)
- **Original Manuals** for older/vintage parts
- **Voice** feedback/motivations (Audible/text readers)
- **Fun** characters, animations & interactions  (Videos, Games, Puzzles explaining systems)
- **Social** interaction/competition (badges, study groups, leader boards) 
- **Bike/Part scanner** (image recognition linked to infos/instuctions)
- **3D animated videos** (clear, size-optimized instuction videos, with multiple soundtracks & subtitles)
- **VR workshop** (allow practical learning without access to expensive bikes/parts)
- **References apps:** [Anki](https://apps.ankiweb.net/), [Quizlet](https://quizlet.com/), [Duolingo](https://www.duolingo.com/), [Brilliant](https://brilliant.org/), [Khan Academy Kids](https://learn.khanacademy.org/khan-academy-kids/), [Unity](https://unity.com/), [Udemy](https://www.udemy.com/)

### Business (Phase II)

 - Online shop
 - Buy/sell used bikes 
 - Buy/sell spare parts to other technicians and customers
 - Buy in bulk as cooperation
 - Bookkeeping/Tax return basic functionality
 - Bike Rental/Fleet management Provide
 - Guarantee Mechanic ratings
 - References: Shopify, WordPress

## Special requirements (for developing countries)

 - Should function on as many **platforms/devices** (old and new)
 - Be **scalable** to many countries and languages. 
 - Function **offline** as much as possible 
 - Using very **small data** files
 - Customizable section **downloads via Wifi**    
 
 ## Business model
 Find solutions enabling it stay free for as many people as possible, while generating enough funds to support continous improvement
